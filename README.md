# The Sandwich

## 游戏规则

- 核心规则：
  - 不是最后一名输，而是 第二名输
  - 第二名需要分别赔付给第一名和第三名

- 回合流程
  - 共 5 回合。前 4 回合为 2 张手牌 + 1 张公共牌，第 5 回合直接比较剩余 3 张手牌。
- 回合积分
  - 第 1 至第 5 回合基础积分依次为 1 / 2 / 3 / 4 / 5。
- 牌型大小
  - 豹子 > 同花顺 > 同花 > 顺子 > 对子 > 单牌
  - `235` 为最小牌型。
- 癞子规则
  - 大小王都是癞子，会自动组成当前能形成的最大牌型。
- 豹子翻倍
  - 只有当 输家是豹子，且它赔付的这个赢家也是豹子时，这一笔赔付才翻倍。
- 并列处理
  - 并列第一：最后一名视为输家。并列第二：两个并列玩家都算输家。三人同牌型：本回合无人输赢。
- 公共牌
  - P1 / P2 / P3 为明牌，P4 为暗牌，并在第 4 回合结算时揭晓。

> 快速示例：
> - 如果三个人分别是 AAA / KKK / QQQ，输的不是 `QQQ`，而是中间的 `KKK`。
> - 如果是 AAA / QQQ / JQK，输家 `QQQ` 赔给 `AAA` 会翻倍，赔给 `JQK` 不翻倍。

## 对局说明

三个人参与的扑克牌游戏;一整副扑克牌(包含大小王);默认情况下底分为 1 分;

- 游戏开始:

发四张公共牌(P1,P2,P3,P4), 其中P1/P2/P3为明牌, P4为暗牌;

逐一发牌, 每个人共 5 张手牌;

第一回合: 每个人出两张手牌, 所有人确定出的牌之后, 共同展示牌型, 与 P1 组合成一个三张牌牌型, 按照大小进行比较, 第二名判负, 其余人获胜;

每个人进行抓牌, 将手牌补充至 5 张;

第二回合: 每个人出两张手牌, 所有人确定出的牌之后, 共同展示牌型, 与 P2 组合成一个三张牌牌型, 按照大小进行比较, 第二名判负, 其余人获胜;

每个人进行抓牌, 将手牌补充至 5 张;

第三回合: 每个人出两张手牌, 所有人确定出的牌之后, 共同展示牌型, 与 P3 组合成一个三张牌牌型, 按照大小进行比较, 第二名判负, 其余人获胜;

每个人进行抓牌, 将手牌补充至 5 张;

第四回合: 每个人出两张手牌, 所有人确定出的牌之后, 共同展示牌型, 然后揭晓 P4 的牌, 每个人的手牌与 P4 组合成一个三张牌牌型, 按照大小进行比较, 第二名判负, 其余人获胜;

第五回合: 每个人出三张手牌, 按照大小进行比较, 第二名判负, 其余人获胜;

- 输赢说明:

第一回合: 输的人需要向其他人支付 1 积分; 若第二名和第三名的牌型一直, 则同时视为第二名, 每个人都需要向第一名支付 1 积分; 第二回合: 输的人需要向其他人支付 2 积分; 若第二名和第三名的牌型一直, 则同时视为第二名, 每个人都需要向第一名支付 2 积分; 第三回合: 输的人需要向其他人支付 3 积分; 若第二名和第三名的牌型一直, 则同时视为第二名, 每个人都需要向第一名支付 3 积分; 第四回合: 输的人需要向其他人支付 4 积分; 若第二名和第三名的牌型一直, 则同时视为第二名, 每个人都需要向第一名支付 4 积分; 第五回合: 输的人需要向其他人支付 5 积分; 若第二名和第三名的牌型一直, 则同时视为第二名, 每个人都需要向第一名支付 5 积分;

- 牌型大小:

豹子>同花顺>同花>顺子>对子>单牌

大小王为癞子牌, 可以组合任意牌型, 组合牌型时, 只能组合最大牌型

所有牌型中 豹子(三个A)最大, 235(不同花色)最小

- 特殊说明:

若输的人为豹子牌型且赢的人也是豹子牌型, 所需支付的积分需要翻倍; 例如: 第一回合中, U1 为 AAA, U2 为 KKK, U3 为 QQQ, 则结果为: U2 需要向 U1 和 U3 分别支付 2 积分 例如: 第五回合中, U1 为 AAA, U2 为 QQQ, U3 为 JQK, 则结果为: U2 需要向 U1 支付 10 积分, 向 U3 支付 5 积分

## 功能描述

- 用户注册：使用邮箱进行注册，输入邮箱后发送验证链接到邮箱中，点击邮箱中的链接跳转到页面：设置用户名和密码
- 用户登录：使用邮箱发送验证码登录或者用户名和密码登录

- 游戏对局：
  - 支持进行中的对局重连
  - 每个回合 30s 出牌时间，若规定时间内未出牌，则系统自动出牌
  - 支持观战
  - 出牌、选牌、结算等固定音效
  - 豹子、同花顺等牌型音效

- 对局记录：
  - 查看已经完成的对局，点击详情之后，显示对局明细

- 对局统计：
  - 对局数
  - 胜率
  - 积分
  - 全胜占比（全胜指同一个对局中5个回合全部胜利）

- 翻倍逻辑：底牌（p1&p2&p3）为某种牌型时，底分翻倍
  - 同花：x2
  - 顺子：x2
  - 同花顺：x3
  - 大小王在P1：x2
  - 大小王在P2：x3
  - 大小王在P3：x4
  - 豹子：x5
